Tool tip/help text/comments
Input or output? timers either measure some input signal or generates an output signal. Sometimes an output generator uses a pin as an input clock.
If generator:
Main function
base rate User enters a frequency, which is mixed with the related input clock which is configured either here or on a clock controller dialog. Doing it here is better, via a "clock source" field, this guy is used to set a prescalar value.
If PWM:
Alignment Active from the start, active after a delay, up/down count and toggle early vs late in synch, i.e. up and late then switch to down and early.
period
sets cyclelength, the rate at which the counter is reloaded when it reaches its terminal value.
duty cycle
sets delay to change from active to inactive
